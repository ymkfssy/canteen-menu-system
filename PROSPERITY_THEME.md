# 🎊 开门红主题使用指南

## ✨ 主题介绍

开门红主题专为节日氛围设计，采用传统的中国红和金色元素，寓意财源滚滚、红红火火。

### 🎨 设计理念

- **红色主调**：象征喜庆、热情、好运
- **金色点缀**：代表财富、尊贵、辉煌
- **渐变效果**：营造层次感和现代感
- **发光特效**：增强节日氛围

### 🌈 色彩方案

#### 背景渐变
```
linear-gradient(135deg, #ff4757 0%, #ff6348 20%, #ffa502 50%, #ff4757 80%, #c44569 100%)
```

#### 分类配色
- **凉菜**：热情红色 `#ff4757`
- **热菜**：温暖橙红 `#ff6348` 
- **主食**：金色主调 `#ffa502`
- **汤品**：粉红柔美 `#ff7675`
- **水果**：甜美粉红 `#fd79a8`

## 🚀 快速开始

### 1. 设置为默认主题

#### 方法一：修改前端默认主题
编辑 `public/index.html`：
```javascript
let currentTheme = 'prosperity'; // 改为 prosperity
```

#### 方法二：后台选择
1. 访问后台管理页面
2. 进入 "主题设置" 标签
3. 点击 "开门红主题" 卡片
4. 点击 "应用主题" 按钮

### 2. 数据库初始化

如果希望新安装的系统默认使用开门红主题：

```bash
# 重新初始化数据库
npx wrangler d1 execute canteen_menu_db --file=database/schema.sql
```

或者在现有系统中执行：
```sql
UPDATE current_menu SET theme = 'prosperity' WHERE id = 1;
```

## 🎨 自定义调整

### 修改颜色
编辑 `public/js/themes.js` 中的 `prosperity` 主题配置：

```javascript
prosperity: {
    name: '开门红',
    colors: {
        primary: 'linear-gradient(135deg, #ff4757 0%, #ff6348 20%, #ffa502 50%, #ff4757 80%, #c44569 100%)',
        // ... 其他颜色配置
    }
}
```

### 调整特效
标题发光效果在 `applyTheme` 函数中定义：

```javascript
@keyframes prosperityGlow {
    0% {
        text-shadow: 0 0 20px rgba(255, 71, 87, 0.9), 0 0 40px rgba(255, 215, 0, 0.6);
    }
    100% {
        text-shadow: 0 0 30px rgba(255, 71, 87, 0.9), 0 0 60px rgba(255, 215, 0, 0.8), 0 0 80px rgba(255, 140, 0, 0.6);
    }
}
```
